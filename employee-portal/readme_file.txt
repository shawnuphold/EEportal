== Description ==

Employee Portal is a comprehensive WordPress plugin designed for businesses to manage their employees, documents, and HR processes securely within WordPress.

Features:
* Secure employee profile management
* Document upload and download (W-2s, I-9s, paystubs)
* Time-off request system
* Company announcements
* Role-based access control
* Mobile-responsive design
* GDPR compliant

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/employee-portal/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Create pages with [employee_login] and [employee_dashboard] shortcodes
4. Configure employees through the Employee Portal admin menu

== Frequently Asked Questions ==

= What file types are supported for document uploads? =
Currently, only PDF files are supported for security reasons.

= Can employees access other employees' documents? =
No, employees can only access their own documents. Only HR managers and administrators can access all employee documents.
